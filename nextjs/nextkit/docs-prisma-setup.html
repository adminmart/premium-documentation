Prisma ORM Quickstart

Complete Prisma Setup Guide with PostgreSQL and MongoDB

Steps to configure Prisma

1. Install Prisma CLI and Client
       
                
npm install prisma --save-dev
npm install @prisma/client
2. Initialize Prisma
       
                
npx prisma init

This creates:

  • prisma/schema.prisma → where your models and DB settings live
  • .env → where you define the database connection URL
3. Setup your desired databse url in .env
       
                
DATABASE_URL="YOUR_DATABASE_URL"
4. Define your data models
       
                
generator client {
  provider = "prisma-client-js"
}

datasource db {
  provider = "mongodb" // or "postgresql"
  url      = env("DATABASE_URL")
}

model User {
  id    String   @id @default(auto()) @map("_id") @db.ObjectId 
  name  String
  email String  @unique
}
5. Push schema to the database
       
                
npx prisma db push   # Use this to sync without migrations 
# OR
npx prisma migrate dev --name init  # Use this to generate a SQL migration
6. Generate Prisma Client
       
                
npx prisma generate
7. Use Prisma Client in your code
       
                
import { NextRequest, NextResponse } from "next/server";
import { PrismaClient } from "@prisma/client";
import bcrypt from "bcryptjs";
